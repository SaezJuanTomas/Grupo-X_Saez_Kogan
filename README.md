# Grupo X

Proyecto de tesis sobre gestión de vulnerabilidades con una interfaz web, una API y automatización de tareas.

## Qué hace

El sistema permite ver, registrar y seguir vulnerabilidades de seguridad de forma ordenada. También incluye una automatización con n8n para traer datos, procesarlos y guardarlos en el backend.

## Partes principales

- Frontend: React + Vite
- Backend: FastAPI
- Base de datos: SQLite
- Automatización: n8n

## Funcionalidades

- Inicio de sesión con roles
- Panel principal con información general
- Lista de vulnerabilidades
- Estadísticas y seguimiento
- Integración con flujos automáticos de n8n

## Estructura general

- frontend/: interfaz web
- backend/: API y base de datos
- n8n_workflow.json: flujo para importar en n8n

## Objetivo del proyecto

El objetivo es mostrar una solución simple, clara y realista para administrar vulnerabilidades y apoyar el trabajo del equipo de seguridad.
