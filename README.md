# Grupo X (Saez & Kogan) - React App

Aplicación web minimalista con React, TypeScript y Vite.

## 🚀 Instalación y ejecución

### Requisitos previos
- Node.js 16+ instalado

### Instalación
```bash
npm install
```

### Desarrollo
```bash
npm run dev
```
La aplicación se abrirá automáticamente en `http://localhost:3000`

### Build para producción
```bash
npm run build
```

### Vista previa del build
```bash
npm run preview
```

## 📁 Estructura del proyecto

```
src/
├── components/        # Componentes reutilizables
│   ├── Encabezado.tsx
│   ├── Encabezado.css
│   ├── MenuLateral.tsx
│   ├── MenuLateral.css
│   ├── ContenidoPrincipal.tsx
│   └── ContenidoPrincipal.css
├── pages/            # Páginas de la aplicación
│   ├── PaginaInicio.tsx
│   └── PaginaInicio.css
├── models/           # Tipos e interfaces TypeScript
│   └── tipos.ts
├── App.tsx           # Componente principal
├── App.css
├── main.tsx
└── index.css
```

## 🎨 Características

- **React 18** con hooks funcionales
- **TypeScript** para seguridad de tipos
- **Vite** para desarrollo rápido
- **Tema claro y minimalista**
- **Responsive design** (mobile-first)
- **Menú hamburguesa** funcional
- Sin dependencias de UI externas

## 📝 Notas

- Toda la interfaz está en español
- Componentes funcionales con TypeScript
- Diseño limpio y simple
- Fácil de extender con nuevas páginas
